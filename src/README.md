# src/

Core VAP implementation.

## Modules

| Module | Description |
|--------|-------------|
| `schema/` | Value ontology definitions and loaders |
| `negotiation/` | Proposal intake, consensus engine, voting |
| `evaluator/` | Runtime alignment scoring against declared values |
| `enforcement/` | Guardrail injection into agent execution pipelines |
| `audit/` | Drift detection, score history, transparency reporting |
